# [643. 子数组最大平均数 I](https://leetcode-cn.com/problems/maximum-average-subarray-i/)

**示例 1:**

```
输入: [1,12,-5,-6,50,3], k = 4
输出: 12.75
解释: 最大平均数 (12-5-6+50)/4 = 51/4 = 12.75
```

**注意:**

1. 1 <= `k` <= `n` <= 30,000。
2. 所给数据范围 [-10,000，10,000]。
